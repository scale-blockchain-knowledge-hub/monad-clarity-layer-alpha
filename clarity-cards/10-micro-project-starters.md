# Micro Project Starters (Hackathon-Friendly)

These starter ideas help builders ship fast.

---

## 1. Parallel Leaderboard Engine
Each user updates their own score; conflicts are minimal.  
Great fit for real-time games or fitness apps.

---

## 2. Microtransaction Paywall
Each user pays individually → ideal for parallelization.  
Works for content platforms, learning apps, or games.

---

## 3. Flash Auctions
Multiple auctions running in parallel.  
Low conflict, high UX benefit.

---

## 4. Batch Verifier Helper
Useful for projects that need compute-heavy, state-light logic.

---

## 5. L2-Style Token Faucet with Rate Limits
Simple to build, useful for hackathon teams.
